# FluidSimulation
Simple fluid simulation

# Additional libraries
SFML

GLM

# Build
```sh
mkdir build
```
```sh
cd build
```
MacOS:
```sh
cmake -G "Xcode" ..
```
Windows:
```sh
cmake -G "Visual Studio" ..
```
Linux: 
```sh
cmake ..
```
